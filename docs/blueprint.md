# **App Name**: TTR Manager

## Core Features:

- Secure Authentication: User authentication and role management using Firebase Authentication with OTP.
- Data Synchronization: Real-time data synchronization and storage of reservations and financial data using Firebase Realtime Database.
- Reservation Management: Management of hotel bookings (add, modify) with admin/employee role-based restrictions.
- Expense Tracking: Tracking of financial transactions (history of expenses), with deletion restricted to administrators.
- Activity Journal: Activity log to monitor all data modification actions with timestamps.
- Theme Toggle: Switch between light and dark themes, to suit different user environments.

## Style Guidelines:

- Primary color: Dark blue (#2E3192) for a professional and modern feel.
- Background color: Light gray (#E0E5EC) in light mode; Dark gray (#333333) in dark mode to reduce eye strain.
- Accent color: Light blue (#3498DB) for interactive elements and highlights to provide clear call to actions.
- Font pairing: 'Poppins' (sans-serif) for headlines and 'PT Sans' (sans-serif) for body text.  Note: currently only Google Fonts are supported.
- Use flat, minimalist icons in a single consistent style that reflect hotel and management functions. Ensure icons are clear and intuitive.
- Clean and structured layout with a focus on ease of use. Utilize grid systems and card-based layouts to organize content effectively.  Ensure key information is easily accessible.
- Subtle transitions and animations for a smooth user experience. For instance, fade-in effects when loading new data or modal transitions. Keep animations brief and purposeful.
